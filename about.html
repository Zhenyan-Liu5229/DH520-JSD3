About the website

This website hosts all the assignments and projects done for DH520:Javascript and D3 class offered at University of Alberta.

The Interactive Story was a practice to learn and manipulate the DOM tree by JavaScript. It was a group project and I partnered with Carolina Pepe to write all of the contents, including HTML, CSS and Javascript. I want to thank her for her help on CSS and we have put our code for the story here.

The Data Visualization project was an attempt to learn and utilize D3. I turned to D3 v4 because I found it more intuitive and easier to follow the available tutorials than v5. And thanks to COVID-19, we didn't have to write the accompanying 1500-word essay.
